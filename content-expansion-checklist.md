# 廣澤外貿培訓頁面補強清單

更新日期：2026-04-22

## 目標

把目前偏向「速查筆記」的內容，補成更接近「新人可照著做」的培訓頁面。

## 優先級 A：可立即補上的通用內容

- `Incoterms 2020`
  - 增加 `FOB / CIF / DAP / DDP / FCA` 的責任拆解表。
  - 增加「什麼情況選哪個條款」的判斷方式。
  - 增加集裝箱、空運、L/C 情境下的常見誤用提醒。
  - 增加 1 個實務案例，讓新人知道風險點在哪裡。

- `付款方式`
  - 增加 `T/T、L/C、D/P、OA` 的實際流程。
  - 增加收款前後的風控檢查點。
  - 增加新客戶、大額訂單、老客戶三種付款建議。
  - 增加「何時不能出貨」的紅線提醒。

- `單據處理`
  - 增加單據時間線，說清楚每份文件何時開立。
  - 增加 `CI / PL / B/L` 常見錯誤清單。
  - 增加單據責任分工，區分業務、工廠、貨代、銀行。
  - 增加交單前 final check checklist。

- `詢報價技巧`
  - 增加 RFQ 資料完整性 checklist。
  - 增加一個報價成本拆解範例。
  - 增加報價跟進節奏與客戶回覆腳本。
  - 增加不能亂承諾的項目，例如交期、模具、索賠、付款條件。

- `品質管理`
  - 補上業務遇到客訴時的第一時間處置步驟。
  - 補上 PPAP / 8D 對業務的實際意義，不只講術語。
  - 增加客訴回覆的資料蒐集清單。
  - 增加暫時對策與永久對策的差別說明。

- `物流操作`
  - 補上出貨前 checklist。
  - 增加截關、爆艙、延船、查驗、提單錯誤等異常處理。
  - 增加 `FCL / LCL / 空運` 的選擇邏輯。
  - 增加和貨代對接時應確認的資料欄位。

- `談判技巧`
  - 補上真實場景話術，而不是只有原則。
  - 增加讓步節奏與交換條件的示例。
  - 補上 `價格 / 付款 / 交期 / 索賠` 四類常見談判應對。

- `跨文化溝通`
  - 補上郵件語氣、會議節奏、時區安排的實用建議。
  - 增加「避免刻板印象」提醒，強調先觀察個體而非只看國別。
  - 增加跨文化會議前的準備 checklist。

- `商務郵件`
  - 增加更多高頻模板：報價回覆、補資料、樣品寄出、交期延遲、L/C 修改、尾款催收。
  - 增加主旨行寫法範例。
  - 增加附件命名和回信時限建議。

- `進階知識`
  - 增加出貨前合規檢查清單。
  - 增加「此章節具時效性，報價或出貨前需再次確認」提醒。
  - 強化 `HS Code / 原產地 / 關稅 / 客戶合規問卷` 的實務關聯。

## 優先級 B：需要公司內部資料才能補的內容

- `廣澤公司知識`
  - 補上真實產能、樣品交期、量產交期、MOQ。
  - 補上設備清單、製程能力、表面處理能力。
  - 補上主要出口市場、代表產品、典型客戶類型。
  - 補上認證明細、品質指標、交付表現。
  - 補上工廠簡介的固定版本，讓業務可直接複製使用。

- `對外介紹數據`
  - 需要內部確認後才能公開：不良率、準交率、客戶數、年產能、模具開發速度、樣品交期。
  - 若沒有對外公開授權，頁面應只保留欄位模板，不直接寫死數字。

## 這次修改策略

- 先補 `A 類內容`，讓網站從概念型內容變成操作型內容。
- `B 類內容` 不造假，先補成「應收集欄位 + 使用提醒」。
- 保持原本章節架構不變，避免影響使用者已熟悉的導航。
- 先強化學習頁內容；練習題庫若有需要，再第二輪補題。

## 這次實作的優先順序

1. 補每章最缺的 `流程 / checklist / 範例 / 異常處理`。
2. 在公司介紹章節加入 `待補真實數據欄位模板`。
3. 在進階章節加入 `時效性提醒`，降低過時資訊風險。
